## Description

This repo renders a lightning strike using CPU ray tracing, volumetric scattering, and emissive geometry

## Project structure

```
COMP4009PROJECT/
├── src/            ← source code directory
│   ├── (… .cpp / .h
├── out/           ← renders
├── README.md       ← this file
```

## Dependencies / Requirements

- Visual Studio
- openFrameworks

## Build and Run

1. Clone / download the repository.  
2. Navigate to the project root directory in Visual Studio
3. Press Run
4. See the renders pop up in Out
